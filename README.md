# Engage

## Available Scripts

In the `backend` directory, you can run:

### `mvnw install -DskipTests`
Generates the JAR file

### `docker build -t engage-backend.jar .`
Builds the Docker image

In the root directory, you can run:

### `docker-compose up -d`
Starts the project

### `docker-compose down`
Stops the project
